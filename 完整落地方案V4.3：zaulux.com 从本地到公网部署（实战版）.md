## 整体部署流程（4 步，约 1 小时）

```
你的电脑（D:\AI\zaulux-site\）
    │
    │  git push（上传代码）
    ▼
GitHub（代码仓库，免费）
    │
    │  自动触发构建
    ▼
Vercel（网站托管，免费）
    │
    │  CNAME 域名指向
    ▼
Cloudflare（DNS + CDN + SSL，免费）
    │
    ▼
🌐 www.zaulux.com（公网可访问）
```

## 第二步：把本地代码上传到 GitHub（需要我在你电脑上操作）

### 2.1 安装 Git（如果还没装）

打开命令行，输入：
```bash
git --version
```

如果显示版本号（如 `git version 2.x.x`）→ 已安装，跳过。
如果提示"不是内部命令"→ 去 https://git-scm.com/download/win 下载安装（一路点 Next）。

### 2.2 创建 GitHub 仓库

1. 登录 GitHub → 右上角 **+** 号 → **New repository**
2. 填：
   - Repository name: `zaulux-site`
   - Description: `ZAULUX welding equipment website`
   - 选 **Public**（公开）
   - **不要**勾选 "Add a README file"（因为本地已经有文件了）
3. 点 **Create repository**

创建后会跳到一个页面，上面有几行命令。**不要关掉这个页面**，下一步要用。

### 2.3 上传代码（告诉我就行，我来操作）

> ⚠️ 这一步由 Hermes（我）在你的电脑上执行。你只需要告诉我："可以上传了"。

我会执行以下操作（你不用管细节）：

```bash
# 1. 进入项目目录
cd D:/AI/zaulux-site

# 2. 初始化 Git
git init

# 3. 添加 .gitignore（排除不需要上传的文件）
echo "node_modules/
dist/
.astro/
.env" > .gitignore

# 4. 添加所有文件
git add .

# 5. 提交
git commit -m "ZAULUX website V1.0 - full site ready for deployment"

# 6. 关联 GitHub 仓库
git remote add origin https://github.com/你的用户名/zaulux-site.git

# 7. 推送到 GitHub
git branch -M main
git push -u origin main
```

上传完成后，刷新 GitHub 仓库页面，你应该能看到所有文件出现在上面。

## 第三步：Vercel 部署（5 分钟）

### 3.1 导入项目

1. 登录 https://vercel.com（用 GitHub 账号）
2. 点 **New Project**
3. 在列表中找到 `zaulux-site` → 点 **Import**
4. 不用改任何设置，直接点 **Deploy**
5. 等待 1-2 分钟，出现 🎉 画面 = 部署成功

### 3.2 拿到临时地址

部署完成后，Vercel 会显示一个临时地址，类似：
```
zaulux-site-abc123.vercel.app
```

**在浏览器打开这个地址**，确认网站能正常显示（和 localhost:4321 一样）。

---

## 第四步：绑定域名 + Cloudflare 加速（20 分钟）

### 4.1 Cloudflare 接管域名 DNS

1. 登录 https://dash.cloudflare.com
2. 点 **Add a site** → 输入 `zaulux.com`
3. 选 **Free** 套餐 → Continue
4. Cloudflare 会自动扫描你阿里云上的 DNS 记录 → 确认无误后 Continue
5. Cloudflare 会给你 2 个 **Nameserver 地址**，类似：
   ```
   elsa.ns.cloudflare.com
   guy.ns.cloudflare.com
   ```
6. **复制这 2 个地址**，去阿里云域名控制台：
   - 登录阿里云 → 域名 → 找到 `zaulux.com` → DNS 修改
   - 把原来的 DNS 服务器替换成 Cloudflare 的 2 个地址
   - 保存

> ⚠️ DNS 修改后需要 **1-24 小时**全球生效。Cloudflare 面板显示 ✅ Active 就说明生效了。

### 4.2 配置 DNS 记录

Cloudflare 面板显示 Active 后：

1. 左侧菜单 → **DNS** → **Records**
2. 添加 2 条记录：

| 类型 | 名称 | 目标 | 代理 |
|------|------|------|:----:|
| CNAME | `www` | `zaulux-site-abc123.vercel.app`（你的 Vercel 地址） | 🟠 |
| CNAME | `@` | `zaulux-site-abc123.vercel.app`（同上） | 🟠 |

> 💡 **代理一定要选 🟠 橙色云朵**（Proxied），这样 Cloudflare 才会帮你加速和防护。

### 4.3 Vercel 绑定自定义域名

1. 回到 Vercel → 你的项目 → **Settings** → **Domains**
2. 输入 `www.zaulux.com` → Add
3. 输入 `zaulux.com` → Add（自动跳转到 www）
4. Vercel 会自动签发 SSL 证书（约 1-2 分钟）

### 4.4 验证

在浏览器打开：
- `https://www.zaulux.com` → 应该看到你的网站
- `https://zaulux.com` → 应该自动跳转到 www
