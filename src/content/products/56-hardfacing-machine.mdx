# H frame M/C

**Category:** automation  
**Article ID:** 56  
**Published:** 2024-08-10 11:54:15

<h3>Hardfacing machine for Steel and General Maintenance Workshops</h3>

<p> H-Frame hardfacing machine machine is a flexible, cost-effective weld surfacing machine with a quick return on investment. It can be used in many applications: flat welding, roller welding, spiral welding and welding with several welding heads at the same time. It is available in various sizes, allowing it to be customised to the needs of the workshop.</p>

<p>   <img alt="hardfacing machine" height="349" src="images/DSPIC/hframe-large.jpg" width="466"/></p>

<p>   Desan AUTO control system is installed for all of the H-Frame machines, either in the Touch or Pendant configuration. The control system is designed for fast and easy weld set-up of different components, while maintaining precise control of the whole welding process.</p>



<h2><a href="welding-machine/automation">H ardfacing machine </a>Characteristics</h2>

<ul>
<li>D3 touch system</li>
<li>Five easily adaptable options</li>
<li>Multiple weld pattern: oscillating beads, stringer beads, square waves, chevron shape and spiral welding</li>
</ul>

<ul>

<li>High quality weld deposits</li>
<li>High productivity in manufacturing</li>
<li>Adaptable to any workshop size</li>
</ul>
